# Exercício 3
Mesmo fluxo do exercício 2.

## Create User
- method HTTP (POST)
- path ("/users")
- body
    - id
    - name
    - email
    - password
- response
    - status 201
    - "Cadastro realizado com sucesso"

## Create Product
- method HTTP (POST)
- path ("/products")
- body
    - id
    - name
    - price
    - description
    - imageUrl
- response
    - status 201
    - "Produto cadastrado com sucesso"

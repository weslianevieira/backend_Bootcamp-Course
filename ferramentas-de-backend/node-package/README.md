## O que esperamos que você entenda com esse exercício?

No exercício de hoje, queremos que vocês absorvam bem o uso e aplicação do Node. Para isso, quebramos os conteúdos de aula em pedaços, para que cada conceito seja lido e aplicado. A finalizar este exercício, você terá exercitado:

- A criação de um `package.json`;
- A criação de scripts customizados;
- A execução de programas utilizando o `node`;
- A recepção de parâmetros em comandos customizados.
